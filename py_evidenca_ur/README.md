# Python EvidencaUr

This is a minimal Python version of the `EvidencaUr` project using [Kivy](https://kivy.org/) for the UI and `sqlite3` for data persistence.

## Running

1. Install Kivy in your environment:
   ```bash
   pip install kivy
   ```
2. Run the application:
   ```bash
   python main.py
   ```

On iOS you can run this code in apps like Pythonista or package it using Kivy's iOS toolchain.
